# Gamepad
一个针对与游戏手柄进行升级的应用程序，意思是设置游戏手柄振动、闪烁等功能进行升级设置的应用程序。

这张是应用程序正常界面，用户通过插入游戏手柄，应用程序反馈给用户，显示不同的结果！
## 第二张图
